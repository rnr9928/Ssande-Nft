# 로컬 백엔드 url용 .env 생성

front 폴더안에 .env 만들어주세요

```
REACT_APP_BACK_URL='http://localhost:4000'
```

# 배포서버 url용 .env.production 생성

```
REACT_APP_BACK_URL='배포서버 주소'
```
